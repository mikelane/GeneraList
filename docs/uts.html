Usability Test Script

Test Script:

  1. Ask Alexa to start GeneraList.
  2. Create a list and add items to it.
  3. Load a list and play it.
  4. Delete an existing List.
  5. Go back and forth in a list.
  6. Start creating a list and cancel in between.

Preference Questions:

  1. On a scale of 1 to 5, how easy was this app to use ?
  2. Would you prefer using this app over other text apps ?
  3. Would you prefer using this app over other similar voice apps ?
  4. On a scale of 1 to 5, how willing would you be to pay for getting this app ?
  5. On a scale of 1 to 5, how likely is it for you to recommend this app to a friend ?

Performance Questions:

  1. On an average, how many attempts does a user take to successfully create a list ?
  2. What was the average response time of the app following each command ?
  3. How many times, on an average, does a user invoke cancel while in the process of creating a list ?
  4. How many times barge-in was invoked ?
  5. How many times help was invoked ?
